# 🧮 Calculadora Java

Projeto simples de calculadora desenvolvido em Java, com o objetivo de praticar lógica de programação e conceitos básicos da linguagem.

---

## 🚀 Funcionalidades

* ✔️ Soma
* ✔️ Subtração
* ✔️ Multiplicação
* ✔️ Divisão
* ✔️ Interação via terminal

---

## 🛠️ Tecnologias utilizadas

* Java

---

## 📂 Estrutura do projeto

```
calculadora-java/
 ├── src/
 │    └── Main.java
 ├── .gitignore
 └── README.md
```

---

## ▶️ Como executar

### 1. Clone o repositório

```bash
git clone https://github.com/Zyfix-creator/calculadora-java.git
```

### 2. Acesse a pasta

```bash
cd calculadora-java
```

### 3. Compile o código

```bash
javac src/Main.java
```

### 4. Execute o programa

```bash
java -cp src Main
```
